Què és un repositori en GitHub?

És un sistema per emmagatzemar dades, recursos o tota mena d'informació. També s'utilitza per organitzar i controlar les versions d'un programari. Els usuaris poden penjar el seu programari o les seves mostres per a compartir amb la comunitat.

Com penjar un fitxer al repositori GitHub?

Per penjar un fitxer a GitHub has de seguir els següents passos:

  1. Crear un repositori: Si encara no tens un repositori, pots crear-ne un a GitHub des del teu compte.
  2. Clonar el Repositori: Utilitza la comanda git clone per a copiar el repositori a la teva màquina local.
  3. Actualitzar el repositori local: Executa la comanda git add per afegir els fitxers a l'àrea de preparació i després git commit per a confirmar els canvis.
  4. Penja els canvis: Utilitza git push per enviar els canvis al repositori remot a GitHub.

Assegura't de configurar correctament el teu entorn de GitHub abans de començar.

Espero que aquesta informació us hagi sigut útil per començar a treballar amb els repositoris de GitHub!
